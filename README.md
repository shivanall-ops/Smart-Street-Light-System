# Smart Street Light Energy Waste Detection System

## Project Description

The Smart Street Light Energy Waste Detection System is a Python and Flask-based application developed to manage street light information, monitor ambient light levels, detect energy wastage, and identify faulty street lights. The system provides both a menu-driven console application and a web dashboard for real-time monitoring.

## Features

- Register Street Lights
- View All Street Lights
- Search Street Light by ID
- Update Street Light Status (ON/OFF)
- Detect Energy Wastage
- Identify Faulty Street Lights
- Generate Reports
- Real-Time Web Dashboard
- Ambient Light Monitoring
- System Health Monitoring

## Technologies Used
