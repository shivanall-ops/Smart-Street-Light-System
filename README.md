# Smart Street Light Energy Waste Detection System

## Project Description
This is a Python-based application that helps manage street lights and detect energy wastage. The project uses SQLite for database management and provides a menu-driven interface.

## Features
- Register Street Light
- View All Street Lights
- Search Street Light by ID
- Update Street Light Status (ON/OFF)
- Detect Energy Wastage
- Generate Reports

## Technologies Used
- Python
- SQLite
- Git & GitHub
- Windsurf IDE

## How to Run
1. Clone the repository.
2. Open the project in Windsurf.
3. Run the following command:

```bash
python main.py
```

## Menu Options

1. Register Street Light
2. View Street Light
3. Search Street Light
4. Update Light Status
5. Detect Energy Wastage
6. Generate Report
7. Exit
